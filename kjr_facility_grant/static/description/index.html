KJR Brücke: Einrichtung ↔ Zuschuss

Aus einer geförderten Hausbuchung direkt den Zuschussantrag erzeugen

Diese optionale Brücke verbindet die Einrichtungsbuchung mit der Zuschussverwaltung. Sie installiert sich automatisch, sobald beide Module vorhanden sind – ohne sie bleiben beide Module eigenständig nutzbar.

Funktionsumfang

Verknüpfung

  • Aktion „Zuschussantrag erstellen" direkt aus der Buchung
  • Antrag wird vorbefüllt (Verband, Zeitraum, Teilnehmer, Unterkunftskosten)
  • Förderbezug der Buchung (gefördert / Aktenzeichen) wird gespiegelt

Übersicht

  • Buchung zeigt den verknüpften Antrag (Smart-Button)
  • Antrag zeigt die zugehörigen Einrichtungsbuchungen
  • Verhindert doppelte Antragserzeugung

Auf echtem Odoo 19 installiert und mit automatisierten Tests abgesichert.
